# ML Foundations – 20 Day Intensive

This repository documents my **20-day intensive journey into the foundations of Machine Learning and Data Science**.  
The goal of this project is to build a strong understanding of the **core tools and concepts used in real-world ML workflows**, while maintaining a structured and transparent learning process.

Over the course of 20 days, I will be studying and practicing key topics including:

- NumPy for numerical computing  
- Pandas for data analysis and data manipulation  
- Statistics and mathematical concepts used in machine learning  
- SQL for querying and analyzing structured data  
- Machine Learning fundamentals and model building  

Each topic will be explored through **hands-on notebooks, exercises, and mini projects**, with all progress documented in this repository.

The objective is to:

- Develop practical skills in working with data  
- Understand the mathematical intuition behind machine learning algorithms  
- Build small end-to-end projects that demonstrate applied knowledge  
- Maintain a clear and organized learning portfolio  

This repository will gradually evolve as new concepts, notebooks, and projects are added throughout the 20-day learning process.
